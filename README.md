# ieee-covid-poster-generator
A script to generate posters on the status of Covid-19 for IEEE MACE SB

To run: 

```
cd ieee-covid-poster-generator/

./poster_gen.py [dd-mm-yyyy]
```
eg: 
```
./poster_gen.py 
```
 or
```
./poster_gen.py 17-05-2020
```
```
